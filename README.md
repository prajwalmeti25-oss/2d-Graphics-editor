# 2D Graphics Editor

## Description

This project is a menu-driven 2D Graphics Editor developed in C using a 2D character array. It allows users to create and manage simple graphical objects on a text-based canvas.

## Features

* Draw Line
* Draw Rectangle
* Draw Triangle
* Draw Circle
* Add Shapes
* Display Canvas
* List Shapes
* Delete Shapes
* Modify Shapes
* Save Drawing to File
* Load Drawing from File

## Technologies Used

* C Programming Language
* Arrays
* Functions
* File Handling

## How to Run

1. Compile the program:
   gcc main.c -o graphics

2. Run the program:
   ./graphics
